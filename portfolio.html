  • Machine Learning, OpenCV

    Gesture-Controlled Chrome Dinosaur Game.

    In this project, I developed a Python application that enables users to control the Google Chrome Dinosaur game using hand gestures. By leveraging computer vision libraries such as OpenCV and PyAutoGUI, the application detects specific hand movements to simulate keyboard inputs. Opening the hand triggers the dinosaur to jump, while closing it (forming a fist) allows the dinosaur to run normally. This innovative approach offers an engaging and interactive way to play the game without traditional input devices.
